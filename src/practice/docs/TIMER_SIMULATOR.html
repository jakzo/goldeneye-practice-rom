What exactly is being simulated?

Full forward input at the maximum 1.25 running boost, NTSC damping (sum = 0.93 × sum + sample, then speed = 0.07 × sum), no strafe, slopes, damage boosts, turning, or collision. The root-motion curve was captured from GoldenEye's real sprint animation on Cradle. For a batch of n timer ticks, the animation advances across all n ticks, one combined per-tick sample is fed through the filter n times, and the final speed is applied for all n ticks—matching the movement structure documented in TIMER.md. Values are calculated in JavaScript doubles from root samples captured to six decimal places.
